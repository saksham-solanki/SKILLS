## Workflow Templates

### Launch New Campaign
1. `ads-plan` — strategy, targeting, budget allocation
2. `ads-creative` — develop creatives and copy
3. Platform-specific skill (e.g., `ads-meta`) — campaign setup
4. `ads-landing` — create landing page
5. `campaign-analytics` — set up tracking

### Campaign Audit
1. `ads-audit` — full performance review
2. `ads-competitor` — benchmark against competitors
3. `ads-budget` — optimize spend allocation
4. Output: action items with priority

### SquareMind Ad Optimization (Live Campaign)
1. Week 1: Learning phase — don't touch
2. Week 2: `ads-audit` — first optimization, pause low performers
3. Week 3-4: Scale winners, `ads-creative` for new test creatives
4. Monthly: `google-ads-report` + `campaign-analytics`
